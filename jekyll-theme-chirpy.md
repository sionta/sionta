## Documentation

How this work, please see https://github.com/cotes2020/jekyll-theme-chirpy

## Prerequisites

- **Ruby** https://www.ruby-lang.org/en/downloads/
- **Git** https://git-scm.com/downloads

## Installation

Run this use `msys64` or `Git Bash`

```bash
# Install bundle
gem install bundle

# Initial bundle
bundle install

# Deploy on local
bundle exec jekyll serve --open-url --livereload
```
